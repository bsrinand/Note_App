# Inkwell

A local note-taking app. Make folders and notes, type or draw with a pen, add images,
light/dark themes. Up to 10 users.

## Run it

1. Install [Node.js](https://nodejs.org).
2. Double-click **`Start Inkwell.bat`**.
3. Open http://localhost:4321 and create your account.

Your notes are saved in the `data/` folder.
